# Dima Zborovskiy Personal Website

## Overview
A premium, minimal one-page personal website for Dima Zborovskiy, an AI and Data Science executive. The site targets enterprise executives and board members seeking senior advisory, board, or fractional leadership roles.

## Project Structure
- `index.html` - Main HTML structure with all content sections
- `styles.css` - Custom CSS with dark theme, purple accents, and smooth animations
- `script.js` - Intersection Observer for scroll animations
- `server.py` - Simple Python HTTP server for development (port 5000)

## Visual Design
- Dark near-black background (#0d0d0f)
- Soft off-white text
- Muted purple accent color (#7c5dbd)
- Inter font family
- Smooth fade-in animations on scroll

## Sections
1. Hero - Name, title, positioning statement, CTA
2. What I Do - Leadership scope summary
3. Selected Impact - Key achievements
4. Experience Snapshot - Career timeline
5. Expertise - Grouped skills and domains
6. Background - Education and credentials
7. Contact - Email link

## Running Locally
The website runs via Python's built-in HTTP server on port 5000:
```bash
python server.py
```

## Deployment
This is a static website suitable for static deployment. All files (HTML, CSS, JS) are served directly.
